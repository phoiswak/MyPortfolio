# Khorommbi Irvin Phosiwa — Portfolio

A personal portfolio website built with React and Vite, showcasing my skills, work experience, education, and contact information.

## Tech Stack

- **React 18** — component-based UI
- **Vite** — fast development server and build tool
- **CSS** — custom styling with CSS variables

## Project Structure

```

├── components/

│   ├── Contact.jsx
```

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Opens at **http://localhost:8080**

### Build for production

```bash
npm run build
```
